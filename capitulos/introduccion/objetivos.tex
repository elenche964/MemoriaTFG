  \subsection{Objetivo General}
  
  El objetivo general de este proyecto es crear un sistema completo con geolocalización haciendo uso de tres dispositivos. Estos dispositivos incluyes un pequeño dispositivo hardware IoT especialmente diseñado para el proyecto a partir de componentes disponibles en el mercado, y dos terminales móviles. Con esto se pretende conseguir una mayor autonomía tanto para el paciente con Alzehimer como para sus cuidadores.
  %Ubicuo, eHealth, servicios
  
  \subsection{Objetivos específicos}
  
  Los objetivos específicos de este proyectos son:
  
  \begin{enumerate}[label=$\bullet$ ]
		\item Hacer uso de los componentes hardware orientados al IoT disponibles en el mercado para construir un dispositivo que se adecúe a nuestras necesidades.
		\item Crear una aplicación especialmente diseñada para personas que sufren EA y su entorno.
		\item Unificar las tecnologías actuales para crear un sistema completo.
  		\item Compartir con la comunidad la información, y los avances recogidos en este proyecto.
  \end{enumerate}
